
Quick Start:

1. Create a new Road Network Object: Main Menu > GameObject > 3D Object > EasyRoads3D > New Road Network

2. From the EasyRoads3D Toolbar in the Inspector select the second tab from the left , the "Road+" tab, and create a new road object

3. Shift + Click in the scene to add marker control points and define the road shape.
   - Markers can be inserted between other markers by moving the mouse to the new marker position and pressing the I key.
   - Shift + R will remove the selected marker.
   - Multiple markers can be selected and edited by holding the Shift key when selecting markers.

4. Open the middle mountain tab when the road is ready and press "Build Terrain's", this will update the terrain to the road shape.


The full online documentation is available here: https://www.easyroads3d.com/v3/manualv3.html
A direct link can be found in the most right Info tab of the EasyRoads3D toolbar in the Inspector
