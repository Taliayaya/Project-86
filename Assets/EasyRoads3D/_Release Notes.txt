v3.2.3f1 release date: 27-07-2023

Version changes according EasyRoads3D Pro v3.2.3f1 that also apply to the free version

Full Release notes:
http://www.easyroads3d.com/releaseNotes.php

For URP and HDRP projects the repective Lit shader can be used. This does not include the terrain z-fighting controls but should work well. And optionally the road network can be raised a little bit higher above the terrain.
The EasyRoads3D Pro package does include a range of additional materials / shaders. Both URP and HDRP support packages are included 


Forum: http://forum.unity3d.com/threads/easyroads3d-v3-the-upcoming-new-road-system.229327/

EasyRoads3D Pro: https://assetstore.unity.com/packages/tools/terrain/easyroads3d-pro-v3-469


Install Notes:

To install this update in an existing EasyRoads3D project it is recommended to follow the next steps:

1. Open a new scene.
2. Simply import package updates over the existing install without first removing the previous package.
3. Open your scene.
